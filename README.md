# Fiskalizacija App

Web aplikacija za unos, pregled i fiskalizaciju računa.

## Tehnologije
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- JSON
- XML

## Funkcionalnosti
- prijava korisnika
- unos računa
- pregled računa
- generiranje XML zahtjeva
- komunikacija s fiskalizacijskim servisom
- pohrana JIR i ZKI podataka
